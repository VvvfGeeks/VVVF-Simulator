1. はじめに

　Next MIDI は、Windows 環境で MIDI 開発を手軽に行うための汎用ライブラリです。本ライブ
ラリを使用すると、たとえば

　・MIDI ファイルから特定のコントロールチェンジを削除する
　・MIDI ファイルを読み込み、特定パートだけを再生する
　・MIDI-IN から入力されたデータを録音し、ファイルに保存する

といった実装を簡単に行うことができます。


2. 構成

　本ライブラリは、主に次のような構成になっています。

　・NextMIDI インターフェース
　　これらのライブラリは、MIDI ファイルや MIDI ポートなどを十分に抽象化し、これらを C# の
　　文法で簡潔に扱うことを可能にします。

　・NextMIDI コアライブラリ
　　これらのライブラリは、Windows API やファイル構造を必要最低限の範囲で抽象化し、C# の
　　文法での自由度の高いプログラミング環境を提供します。

　・Avenue ライブラリ
　　Avenue ライブラリは、Next MIDI の実装を支える基幹ライブラリです。


3. 使い方

　Next MIDI の使用法等については順次 Web に載せて行きますので、そちらをご参照ください。
